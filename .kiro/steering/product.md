# Product Overview

## Kubernetes Learning Game

An interactive, terminal-based educational game that teaches Kubernetes concepts through real-world command execution. Players use genuine kubectl commands against an actual Kubernetes cluster to solve narrative-driven challenges that progressively increase in difficulty.

## Core Value Proposition

Unlike simulations, this game provides hands-on experience with real Kubernetes operations in a safe, isolated environment. Players learn by doing, executing actual kubectl commands while following an engaging story that contextualizes their learning.

## Target Experience

- **Engagement**: 60+ minutes of compelling gameplay content
- **Learning Style**: Progressive difficulty scaling from basic to advanced Kubernetes concepts
- **Inspiration**: Interactive learning platforms like bash crawler and OverTheWire
- **Safety**: Namespace isolation and resource quotas prevent destructive operations

## Key Features

- Real cluster integration with actual kubectl command execution
- Narrative-driven challenges with meaningful story context
- Progressive difficulty that introduces concepts incrementally
- Hint system and educational feedback for continuous learning
- Progress tracking, achievements, and session persistence
- Safe experimentation with cluster reset capabilities
